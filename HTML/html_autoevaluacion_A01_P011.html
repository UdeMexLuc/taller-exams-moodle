Exam 1

  1. Question

    ¿Cuáles de las siguientes afirmaciones describen adecuadamente una doxa?


    1. Busco razones y evidencias para sostener una postura.
    2. Lo normal siempre es lo correcto; así se ha hecho y así debe seguir.
    3. Reviso si mis creencias se sostienen en distintos contextos.
    4. Repito lo que oigo sin revisarlo porque ‘siempre ha funcionado’.

    Solution

    Las opciones que consideran creencias asumidas sin examen crítico.

    Recuerda: La doxa remite a opiniones comunes no cuestionadas; el propósito es comenzar a detectarlas en lo cotidiano para poder problematizarlas.


  2. Question

    Selecciona todas las opciones correctas de acuerdo con el siguiente caso: “En Posit Cloud solo se puede hacer escritos tipo Word.”


    1. Verdadero, Posit Cloud solo permite documentos de texto como Word.
    2. Falso, Posit Cloud es una plataforma pensada principalmente para la ciencia de datos, no solo para textos.
    3. Verdadero, Posit Cloud es un procesador de texto básico en línea.
    4. Verdadero, no se puede ejecutar código en Posit Cloud.

    Solution

    La afirmación inicial es falsa. Posit Cloud (anteriormente RStudio Cloud) es una plataforma en línea diseñada para trabajar principalmente con R y Python, permitiendo escribir código, realizar análisis de datos, crear visualizaciones, generar reportes dinámicos y colaborar en proyectos. No es un simple editor de texto ni está limitado a documentos tipo Word. Es importante reconocer las capacidades reales de las herramientas que usas para aprovecharlas al máximo en tu aprendizaje.
